# Contributing to SOVEREIGN

We are excited that you want to contribute to SOVEREIGN! 🚀

### How to Help
1. **Report Bugs:** Open an issue if you find something broken.
2. **Suggest Features:** Tell us how to make SOVEREIGN even more useful for "Broke-Friendly" users.
3. **Submit PRs:**
   - Fork the repo.
   - Create a feature branch.
   - Run `cargo check` and `cargo test` before submitting.
   - Make sure your code is commented and clean.

### Code Style
We follow standard Rust idioms. Use `cargo fmt` to format your code before pushing.

### Privacy First
Never commit your `sovereign.toml` or any files containing API keys.

Thank you for building the future of autonomous local AI!
